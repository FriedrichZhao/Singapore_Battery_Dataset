## SingaporeBatteryDataset

The dataset is balanced with about 50 images for each battery type. We consider common battery types in Singapore, including:

1. Duracell (alkaline), 
2. IKEA (alkaline), 
3. Energizer (alkaline), 
4. Energizer (industrial), 
5. Energizer (lithium), 
6. Exell (Ni-MH), 
7. Exell (Ni-CD), 
8. GP (alkaline), 
9. Klarus, 
10. Others.
